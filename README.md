# Streaming-Data-Project---Uber-End-to-End-Azure-Data-Engineering-Project
This project focus on real time data processing by utilizing Azure Databricks, Azure Data Factory, Azure Event Hubs, PySpark, Spark Structured Streaming, and Spark Declarative Pipelines. A medallion architecture has been implemented and an effort has been made to create "One Big Table" for operational use. This concept help domains to create their own model. Here a model start schema has been implemented.
